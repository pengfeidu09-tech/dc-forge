# DC Forge

> 传统方案让客户想象未来，我们让客户在签约前先运行未来。

DC Forge 是面向神州数码「AI驱动的客户方案生成引擎」命题开发的比赛原型。

项目尝试将客户的模糊需求转化为结构化业务流程，再生成可执行、可验证、可修改的解决方案，而不是只输出一份静态PPT。

## 项目目标

核心链路：

```text
客户需求
→ ProcessSpec：需求与现状流程结构化
→ SolutionBundle：生成三套候选方案
→ Runtime：执行选定方案
→ RunReport：展示运行结果和价值指标
→ 客户增加约束
→ 重新编译方案并显示变化
